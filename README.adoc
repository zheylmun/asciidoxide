AsciiDoc tooling written in Rust.

== Crates

link:crates/parser/[`asciidoxide-parser`]::
Zero-copy, spec-compliant AsciiDoc parser producing an Abstract Semantic Graph (ASG).
See the link:crates/parser/README.adoc[parser README] for details on testing, project structure, and the TCK harness.

link:crates/lsp/[`asciidoxide-lsp`]::
Language Server Protocol server built on the parser.
Provides real-time diagnostics via https://github.com/ebkalderon/tower-lsp[tower-lsp].
+
[source,bash]
----
cargo install --path crates/lsp
----

== Editor Extensions

link:editors/zed/[Zed]::
Zed editor extension that discovers `asciidoxide-lsp` in PATH and provides AsciiDoc syntax highlighting via https://github.com/cathaysia/tree-sitter-asciidoc[tree-sitter-asciidoc].

== Building

[source,bash]
----
cargo build            # Build parser + LSP (default members)
cargo build --workspace  # Build everything including editor extensions
----
